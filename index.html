Path Block Builder Instructions

How To Start:

To start the game click the "Start Game" button below the game window.

Controls
  • Arrow Keys - Use the keyboard arrow keys to move around the game window and determine where to put the block.
  • Space Bar - Use the space bar to place a block on the canvas.
Game Play

The goal of the game is to cover at least 40% of the canvas (20/48 squares) within the given time limit. If successful you will gain access to the next round. Each round will give you a shorter time limit to cover 40% of the space.

The possible blocks that can be played in order are randomly generated and can found lined up horizontally below the game window. As each block is gets played the line will shift right with new block generated at the end of the line.
